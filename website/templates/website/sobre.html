📽 Sobre o projeto

Este site foi criado como um espaço dedicado ao filme All About Lily Chou-Chou (リリィ・シュシュのすべて, 2001), do diretor japonês Shunji Iwai. O objetivo é apresentar a obra para novos espectadores e oferecer informações sobre seu elenco, contexto histórico e impacto cultural. O site é puramente informativo e sem fins lucrativos.

Fan Site Sem fins lucrativos Cinema Japonês Educacional

🛠 Tecnologias utilizadas

Site construído utilizando Django para o roteamento e arquitetura baseada em templates. No front-end, utilizamos HTML5, CSS3 e referências ao ambiente digital dos anos 2000 que permeiam a narrativa do próprio filme.

Django Python HTML5 CSS3

👤 Autores

Site desenvolvido por Kaio e Álvaro, alunos do curso técnico em Informática para Internet no IFSPP (Campus São Paulo do Potengi). Este projeto reflete a aplicação prática de conceitos de desenvolvimento web integrados ao framework Django, unindo tecnologia e apreciação cinematográfica.

Kaio Álvaro Informática para Internet IFSPP
