👤 Autores

Somos Kaio e Alvaro, estudantes do 3º ano de Informática para Internet no IFSPP (Campus São Paulo do Potengi). Enquanto o Kaio focou em capturar a estética visual e o "vibe" do site para que parecesse um fórum antigo do filme, o Alvaro cuidou da arquitetura e da organização do conteúdo. Estamos prestes a nos formar e este site é uma das nossas formas de deixar nossa marca unindo tecnologia e arte.

Kaio - Design Alvaro - Dev 3º Info Web IFSPP

📽 Sobre o projeto

Este site nasceu de uma obsessão compartilhada pela atmosfera melancólica de All About Lily Chou-Chou. O que começou como uma discussão em uma aula de web design sobre a estética dos anos 2000 se tornou este tributo digital. Nosso objetivo foi criar um porto seguro para outros fãs brasileiros encontrarem informações sobre a obra-prima de Shunji Iwai, unindo nossa paixão pelo cinema alternativo com o que aprendemos em sala de aula.

Projeto Pessoal Fãs de Iwai Cinema Japonês Tributo ao Éter

🛠 Tecnologias utilizadas

Para dar vida a este projeto, utilizamos o poder do Django no back-end para gerenciar a estrutura das páginas. O front-end foi lapidado com HTML5 e CSS3 puro, buscando replicar a sensação visual fria e, ao mesmo tempo, acolhedora que as músicas da Lily Chou-Chou transmitem. Foi nosso maior desafio prático até agora no curso.

Django Framework Python Frontend Minimalista Info Web
